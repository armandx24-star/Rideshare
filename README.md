# Rideshare
A full-stack ride-sharing web application that allows users to book rides, drivers to accept ride requests, and admins to manage the platform efficiently.  This project replicates the core functionality of popular ride-hailing platforms like Ola and Uber, including real-time booking, fare calculation, and user-driver interaction.

🔥 Features:
	•	User Registration & Login (OTP/Auth system)
	•	Book Ride (Pickup & Drop location)
	•	Real-time Driver Matching
	•	Fare Estimation System
	•	Ride History for Users & Drivers
	•	Multiple Payment Options (Cash / UPI / Online)
	•	Driver Dashboard (Accept/Reject rides)
	•	Admin Panel (Manage users, drivers, rides)
	•	Responsive UI (Mobile + Desktop)

  🛠️ Tech Stack:
	•	Frontend: HTML, CSS, JavaScript (or React if used)
	•	Backend: Node.js / PHP / Python (edit according to your project)
	•	Database: MySQL / MongoDB
	•	APIs: Google Maps API (for location & distance calculation)

🎯 Purpose:
This project is developed for learning full-stack development and understanding real-world ride-sharing system architecture.
